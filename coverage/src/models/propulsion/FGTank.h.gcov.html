      65             : /** Models a fuel tank.
      66             : 
      67             : <h3>Fuel Temperature:</h3>
      68             :  
      69             :     Fuel temperature is calculated using the following assumptions:
      70             : 
      71             :     Fuel temperature will only be calculated for tanks which have an initial
      72             :     fuel temperature specified in the configuration file.
      73             : 
      74             :     The surface area of the tank is estimated from the capacity in pounds.  It
      75             :     is assumed that the tank is a wing tank with dimensions h by 4h by 10h. The
      76             :     volume of the tank is then 40(h)(h)(h). The area of the upper or lower 
      77             :     surface is then 40(h)(h).  The volume is also equal to the capacity divided
      78             :     by 49.368 lbs/cu-ft, for jet fuel.  The surface area of one side can then be
      79             :     derived from the tank's capacity.  
      80             : 
      81             :     The heat capacity of jet fuel is assumed to be 900 Joules/lbm/K, and the 
      82             :     heat transfer factor of the tank is 1.115 Watts/sq-ft/K.
      83             : 
      84             : <h3>Fuel Dump:</h3>
      85             : 
      86             :     Fuel dumping is handled by the FGPropulsion class.  A standpipe can be
      87             :     defined here for each tank which sets the level of contents (in pounds)
      88             :     which is not dumpable.  Default standpipe level is zero, making all contents
      89             :     dumpable.
      90             : 
      91             : <h3>Fuel Transfer:</h3>
      92             : 
      93             :     Fuel transfer is handled by the FGPropulsion class, however the contents of
      94             :     tanks may be manipulated directly using the SetContents() function here, or
      95             :     via the property tree at <tt>propulsion/tank[i]/contents-lbs</tt>, where i
      96             :     is the tank number (Tanks are automatically numbered, starting at zero, in
      97             :     the order in which they are read in the aircraft configuration file).  The
      98             :     latter method allows one to use a system of FCS components to control tank
      99             :     contents.
     100             : 
     101             :     There is also a property
     102             :     <tt>propulsion/tank[i]/external-flow-rate-pps</tt>. Setting this property to
     103             :     a positive value causes the tank to fill at the rate specified.  Setting a
     104             :     negative number causes the tank to drain. The value is the rate in pounds of
     105             :     fuel per second. The tank will not fill past 100% full and will not drain
     106             :     below 0%.  Fuel may be transfered between two tanks by setting the source
     107             :     tank's external flow rate to a negative value and the destination's external
     108             :     flow rate to the same positive value.  Care must be taken to stop fuel flow
     109             :     before the source tank becomes empty to prevent phantom fuel being created.
     110             : 
     111             : <h3>Configuration File Format:</h3>
     112             : 
     113             : @code
     114             : <tank type="{FUEL | OXIDIZER}">
     115             :   <grain_config type="{CYLINDRICAL | ENDBURNING | FUNCTION}">
     116             :     <length unit="{IN | FT | M}"> {number} </length>
     117             :     <bore_diameter unit="{IN | FT | M}"> {number} </bore_diameter>
     118             :     [<ixx unit="{IN | FT | M}"> {function} </ixx>
     119             :     <iyy unit="{IN | FT | M}"> {function} </iyy>
     120             :     <izz unit="{IN | FT | M}"> {function} </izz>]
     121             :   </grain_config>
     122             :   <location unit="{FT | M | IN}">
     123             :     <x> {number} </x>
     124             :     <y> {number} </y>
     125             :     <z> {number} </z>
     126             :   </location>
     127             :   <drain_location unit="{FT | M | IN}">
     128             :     <x> {number} </x>
     129             :     <y> {number} </y>
     130             :     <z> {number} </z>
     131             :   </drain_location>
     132             :   <radius unit="{IN | FT | M}"> {number} </radius>
     133             :   <capacity unit="{LBS | KG}"> {number} </capacity>
     134             :   <inertia_factor> {number:0-1} </inertia_factor>
     135             :   <contents unit="{LBS | KG}"> {number} </contents>
     136             :   <temperature> {number} </temperature> <!-- must be degrees fahrenheit -->
     137             :   <standpipe unit="{LBS | KG"}> {number} </standpipe>
     138             :   <unusable unit="{GAL | LTR | M3 | IN3 | FT3 | CC}"> {number} </unusable>
     139             :   <priority> {integer} </priority>
     140             :   <density unit="{KG/L | LBS/GAL}"> {number} </density>
     141             :   <type> {string} </type> <!-- will override previous density setting -->
     142             : </tank>
     143             : @endcode
     144             : 
     145             : <h3>Definition of the tank configuration file parameters:</h3>
     146             : 
     147             : - \b type - One of FUEL or OXIDIZER.  This is required.
     148             : - \b radius - Equivalent radius of tank for modeling slosh, defaults to inches.
     149             : - \b grain_config type - One of CYLINDRICAL or ENDBURNING.
     150             : - \b length - length of tank for modeling solid fuel propellant grain, defaults
     151             :               to inches.
     152             : - \b capacity - Capacity, defaults to pounds.
     153             : - \b contents - Initial contents, defaults to pounds.
     154             : - \b temperature - Initial temperature, defaults to degrees Fahrenheit.
     155             : - \b standpipe - Minimum contents to which tank can dump, defaults to pounds.
     156             : - \b unusable - Contents that cannot be used for combustion in the engine,
     157             :                 defaults to gallons.
     158             : - \b priority - Establishes feed sequence of tank. "1" is the highest priority.
     159             : - \b density - Density of liquid tank contents.
     160             : - \b type - Named fuel type. One of AVGAS, JET-A, JET-A1, JET-B, JP-1, JP-2,
     161             :             JP-3, JP-4, JP-5, JP-6, JP-7, JP-8, JP-8+100, RP-1, T-1, ETHANOL,
     162             :             HYDRAZINE, F-34, F-35, F-40, F-44, AVTAG, AVCAT
     163             : 
     164             : location:
     165             : - \b x - Location of tank on aircraft's x-axis, defaults to inches.
     166             : - \b y - Location of tank on aircraft's y-axis, defaults to inches.
     167             : - \b z - Location of tank on aircraft's z-axis, defaults to inches.
     168             : 
     169             : drain_location:
     170             : - \b x - Location of tank drain on aircraft's x-axis, defaults to inches.
     171             : - \b y - Location of tank drain on aircraft's y-axis, defaults to inches.
     172             : - \b z - Location of tank drain on aircraft's z-axis, defaults to inches.
     173             : 
     174             : <h3>Default values of the tank configuration file parameters:</h3>
     175             : 
     176             : - \b type - ttUNKNOWN  (causes a load error in the propulsion configuration)
     177             : - \b location, \b drain_location - both optional, but a warning message will
     178             : be printed to the console if the location is not given
     179             : - \b x - 0.0  (both full and drained CG locations)
     180             : - \b y - 0.0  (both full and drained CG locations)
     181             : - \b z - 0.0  (both full and drained CG locations)
     182             : - \b radius - 0.0
     183             : - \b capacity - 0.00001 (tank capacity must not be zero)
     184             : - \b contents - 0.0
     185             : - \b temperature - -9999.0 (flag which indicates no temperature is set)
     186             : - \b standpipe - 0.0 (all contents may be dumped)
     187             : - \b unusable - 0.0 (all contents may be burnt)
     188             : - \b priority - 1 (highest feed sequence priority)
     189             : - \b density - 6.6
     190             : 
     191             :     @author Jon Berndt, Dave Culp
     192             :     @see Akbar, Raza et al. "A Simple Analysis of Fuel Addition to the CWT of
     193             :     747", California Institute of Technology, 1998,
     194             :     http://www.galcit.caltech.edu/EDL/projects/JetA/reports/lumped.pdf
     195             : */
